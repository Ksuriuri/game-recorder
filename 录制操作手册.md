# 录制操作说明

给使用 **game-recorder-portable-*.zip** 压缩包的同学。解压后按下面做就行。

---

## 怎么用（4 步）

1. **解压**到 **D 盘**（别放 C 盘，网吧重启 C 盘会被清空）
2. 双击 **`install.bat`**，等十几秒（只需第一次做）
3. 双击 **`run.bat`**，右上角会出现小窗口
4. 进游戏 → **连按两次大写键（Caps Lock）** 开始录 → 再连按两次停止 → 点小窗口 **「退出」** 关程序

录好的文件在文件夹 **`recordings`** 里。

---

## 录制时怎么操作

**可以做的：**

- 用 **W、A、S、D** 控制人物走路
- 转鼠标看方向（鼠标点击不算「乱按键」）
- **按住** WASD 慢慢走，别松手太久

**不要做的：**

- 别按 **WASD 以外的键**（空格、Shift、Esc、数字键等都会自动停录）
- 别 **超过 10 秒不按 WASD**（只转鼠标不算走路）
- 别 **疯狂连按 WASD 或猛晃鼠标**（连续 1 秒会停录）
- 单次录制 **少于 10 秒** 的数据会被丢掉

想结束一段：最好 **连按两次大写键** 手动停，最稳。

自动停了会在屏幕中间弹红字提示；再 **连按两次大写键** 可以开始下一段，红字会消失。

---

## 几个要注意的

| 事项 | 说明 |
|------|------|
| 游戏窗口 | 用 **无边框窗口**，别用独占全屏（否则小窗口可能看不到） |
| 声音 | 不用额外设置；Windows **默认扬声器** 要对（别选关掉的 HDMI） |
| 换耳机 | 先停录，换好再开新一段；录着录着换可能后半段没声 |
| 关程序 | 点右上角 **「退出」**；别在任务管理器里强关，可能丢数据 |
| 又双击 run.bat | 会自动关掉旧的再开新的，录着别重复点 |

游戏卡的话，可以改用：`run.bat --fps 20 --quality 28 --x264-threads 1`

---

## 没声音怎么办

1. 确认 Windows **声音输出** 选对了（能听到游戏声就行）
2. 确认游戏本身没静音
3. 还不行，找管理员用 `run.bat --console` 看日志

---

## 一句话记住

> **install 一次 → run 启动 → 连按两次大写键开/关 → 只按 WASD 走路 → 点退出关程序。**
